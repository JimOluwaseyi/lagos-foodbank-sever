# Getting Started
git clone https://github.com/JimOluwaseyi/lagos-foodbank-sever.git

cd lagos-foodbank-server

## Setup

#### Install gems with:

bundle install

### Setup database with:

* rails db:create
* rails db:migrate 

# Usage
### Start server with:

* rails s
* This will Open http://localhost:3000/ in your browser.


# Deployment instructions
